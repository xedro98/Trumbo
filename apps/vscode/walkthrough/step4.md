# Unlock Specialized Capabilities with MCP

**The Model Context Protocol (MCP) connects Trembo to a world of external tools and data.**

Trembo isn't limited to your local code. With the Model Context Protocol, it can reach into databases, APIs, and any MCP-compatible server to pull in the context a task actually needs. Configure local or remote MCP servers to give Trembo specialized, secure tools for real-world development work.

Trembo is fully open source and self-hostable — bring your own API keys, run it anywhere, and keep full control of your workflow.
