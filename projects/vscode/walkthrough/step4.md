# Unlock Specialized Capabilities with MCP

**The Model Context Protocol (MCP) connects Trumbo to a world of powerful tools.**

Go beyond local code. With the Model Context Protocol (MCP), Trumbo accesses vital context from external datasources like databases and APIs. Configure local or remote MCP servers to give Trumbo specialized, secure tools for complex, real-world development tasks.
