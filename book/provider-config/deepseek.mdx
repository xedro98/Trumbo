### Get API Credentials

1.  **Sign Up/Sign In:** Go to the [DeepSeek Platform](https://platform.deepseek.com/) and create an account or sign in.
2.  **Navigate to API Keys:** Open the [API keys](https://platform.deepseek.com/api_keys) section.
3.  **Create a Key:** Click "Create new API key" and give it a recognizable name (e.g., "Trumbo").
4.  **Copy the Key:** Copy the key right away ‚ÄĒ you won't be able to view it again ‚ÄĒ and store it somewhere safe.

### Configuration in Trumbo

> See [Authorization & Model Selection](/getting-started/authorizing-with-trumbo#menu).

1.  **Open Trumbo Settings:** Click the ‚öôÔłŹ icon in the Trumbo panel.
2.  **Select Provider:** Choose "DeepSeek" from the "API Provider" dropdown.
3.  **Enter API Key:** Paste your DeepSeek API key into the "DeepSeek API Key" field.
4.  **Select Provider Model:** Pick the model you want from the "Model" dropdown.
