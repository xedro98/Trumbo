Google Gemini is Google's family of multimodal AI models, offering some of the largest context windows available and strong performance across coding, reasoning, and document analysis.

**Website:** [https://ai.google.dev/](https://ai.google.dev/)

### Get API Credentials

1.  **Sign Up/Sign In:** Go to [Google AI Studio](https://aistudio.google.com/) and sign in with your Google account.
2.  **Get API Key:** Open [aistudio.google.com/apikey](https://aistudio.google.com/apikey).
3.  **Create a Key:** Click "Create API Key" and select or create a Google Cloud project.
4.  **Copy the Key:** Copy the key immediately and store it securely.

### Configuration in Trumbo

> See [Authorization & Model Selection](/getting-started/authorizing-with-trumbo#menu).

1.  **Open Trumbo Settings:** Click the settings icon (‚öôÔłŹ) in the Trumbo panel.
2.  **Select Provider:** Choose "Google Gemini" from the "API Provider" dropdown.
3.  **Enter API Key:** Paste your Google AI API key into the "Gemini API Key" field.
4.  **Select Provider Model:** Pick the model you want from the "Model" dropdown.

### Tips and Notes

-   **Pricing:** Refer to the [Google AI pricing page](https://ai.google.dev/gemini-api/book/models/gemini) for the latest information.
