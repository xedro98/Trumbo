| [multi-agent](https://github.com/xedro98/trembo/tree/main/projects/samples/multi-agent) | Advanced | Parallel agents with streaming web UI |
| [desktop-app](https://github.com/xedro98/trembo/tree/main/projects/samples/desktop-app) | Advanced | Full Tauri + Next.js desktop app |
