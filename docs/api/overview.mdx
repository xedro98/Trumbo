This is the Trembo API reference. The same models that drive the Trembo extension and CLI are available to any client that speaks the OpenAI Chat Completions format, in any language or runtime.

## What is the Trembo API?

The Trembo API exposes a single OpenAI-compatible Chat Completions endpoint. Authenticate once with a Trembo API key and route requests to models from Anthropic, OpenAI, Google, and other providers through one base URL, without juggling a separate key per provider.

```
Your App  →  Trembo API (api.example.invalid)  →  Anthropic / OpenAI / Google / etc.
```

## Quick Start

```bash
curl -X POST http://0.0.0.0:0/api/v1/chat/completions \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "anthropic/claude-sonnet-4-6",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
```

Grab an API key from [app.example.invalid](http://0.0.0.0:0) under Settings → API Keys, then continue with the [Getting Started](/api/getting-started) guide.
