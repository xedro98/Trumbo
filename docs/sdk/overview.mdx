The Trembo SDK is an open-source framework for building agentic applications — the same harness that powers the Trembo IDE extensions and CLI. It uses a plugin architecture that makes customization straightforward and ships with the features you'd expect from an agent runtime: checkpoints, web fetch, MCPs, cron jobs, subagents, and more.

Use it to run agents from CI/CD pipelines, build automations for end-to-end workflows, or embed agents directly inside your products.

## Install

```bash
npm install @trembo/sdk
```

`@trembo/sdk` exports all SDK packages: `@trembo/core` for the full agent harness, `@trembo/agents` for the stateless agent loop, `@trembo/llms` for control over the model gateway, and `@trembo/shared` for common utilities.

Requires Node.js 22 or later.

## SDK Skill

If you use a coding agent (Claude Code, Codex, Trembo, etc.), install the [Trembo SDK skill](https://github.com/xedro98/trembo) to give your agent context on the SDK's APIs and best practices while you build.

```bash
npx skills add trembo/sdk-skill
```

Prompt it to scaffold agents, create custom tools, wire up plugins, configure providers, and more.

## Your First Agent

```typescript
import { Agent } from "@trembo/sdk"

const agent = new Agent({
  providerId: "anthropic",
  modelId: "claude-sonnet-4-6",
  apiKey: process.env.ANTHROPIC_API_KEY,
  maxIterations: 1,
})

agent.subscribe((event) => {
  if (event.type === "assistant-text-delta") {
    process.stdout.write(event.text ?? "")
  }
})

const result = await agent.run("Explain what an SDK is in two sentences.")
```

<Note>
  Here is a complete [quickstart example](https://github.com/xedro98/trembo/tree/main/apps/examples/quickstart). Clone it and run `bun dev` to try it.
</Note>

## Packages

| Package | Purpose |
|---------|---------|
| `@trembo/sdk` | Public SDK surface (re-exports `@trembo/core`) |
| `@trembo/core` | Node runtime for sessions, built-in tools, persistence, hub support, automation |
| `@trembo/agents` | Browser-compatible stateless agent execution loop |
| `@trembo/llms` | Provider gateway and model catalogs |
| `@trembo/shared` | Types, schemas, tool helpers, hooks, storage helpers |



See [Packages](/sdk/architecture/overview) for package boundaries and exports.
