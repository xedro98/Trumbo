Working examples live in the [SDK repository](https://github.com/xedro98/trembo/tree/main/apps/examples), organized by difficulty:

| Example | Difficulty | Description |
|---------|------------|-------------|
| [quickstart](https://github.com/xedro98/trembo/tree/main/apps/examples/quickstart) | Beginner | Send one prompt, stream the response (~15 lines) |
| [cli-agent](https://github.com/xedro98/trembo/tree/main/apps/examples/cli-agent) | Beginner | Interactive terminal chat with a shell tool |
| [code-review-bot](https://github.com/xedro98/trembo/tree/main/apps/examples/code-review-bot) | Intermediate | AI code reviewer with custom tools and structured output |
| [multi-agent](https://github.com/xedro98/trembo/tree/main/apps/examples/multi-agent) | Advanced | Parallel agents with streaming web UI |
| [desktop-app](https://github.com/xedro98/trembo/tree/main/apps/examples/desktop-app) | Advanced | Full Tauri + Next.js desktop app |
