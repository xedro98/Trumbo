Welcome to the Trembo documentation. Whether you're setting up for the first time or pushing into advanced capabilities, everything you need is here.

## What is Trembo?

Trembo is an AI coding agent that lives in your terminal. It reads and writes files, runs terminal commands, drives a browser, and helps you build features through natural conversation. Every action waits for your explicit approval, so you stay in control the whole way through.

## Editor integration

The Trembo CLI also integrates with ACP-capable editors (such as **Zed** and **Neovim**) via `trembo --acp`.
