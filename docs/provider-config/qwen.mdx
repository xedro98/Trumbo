### Get API Credentials

1.  **Sign Up/Sign In:** Go to the [Alibaba Cloud Bailian Console](https://bailian.console.aliyun.com/) and create an account or sign in.
2.  **Navigate to API Keys:** Open the API key management section.
3.  **Create a Key:** Generate a new API key for your application.
4.  **Copy the Key:** Copy the key immediately and store it securely.

### Configuration in Trembo

> See [Authorization & Model Selection](/getting-started/authorizing-with-trembo#menu).

1.  **Open Trembo Settings:** Click the settings icon (⚙️) in the Trembo panel.
2.  **Select Provider:** Choose "Alibaba Qwen" from the "API Provider" dropdown.
3.  **Select Region:** Choose "International" or "China" based on your location.
4.  **Enter API Key:** Paste your Qwen API key into the "Qwen API Key" field.
5.  **Select Provider Model:** Pick the model you want from the "Model" dropdown.

### Tips and Notes

-   **Vision Support:** VL (Vision-Language) models support image inputs for multimodal tasks.
-   **Pricing:** Check the [Alibaba Cloud Bailian Console](https://bailian.console.aliyun.com/) for current pricing.
