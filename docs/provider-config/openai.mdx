Trembo can reach models directly through the official OpenAI API.

## Choose an OpenAI Path

- **[OpenAI API (key-based)](#get-api-credentials):** Use your OpenAI API key.
- **[OpenAI Codex (Subscription OAuth)](#openai-codex-oauth):** Sign in with your OpenAI account, using your OAI subscription (no key management).

**Website:** [https://openai.com/](https://openai.com/)

## OpenAI API (key-based)

### Get API Credentials

1.  **Sign Up/Sign In:** Visit the [OpenAI Platform](https://platform.openai.com/). Create an account or sign in if you already have one.
2.  **Navigate to API Keys:** Once logged in, open the [API keys section](https://platform.openai.com/api-keys).
3.  **Create a Key:** Click "Create new secret key". It's good practice to give it a descriptive name (e.g., "Trembo API Key").
4.  **Copy the Key:** Copy the generated key immediately — for security reasons, OpenAI won't show it again — and store it somewhere safe.

### Configuration in Trembo

> See [Authorization & Model Selection](/getting-started/authorizing-with-trembo#menu).

1.  **Open Trembo Settings:** Click the settings gear icon (⚙️) in the Trembo panel.
2.  **Select Provider:** Choose "OpenAI" from the "API Provider" dropdown.
3.  **Enter API Key:** Paste your OpenAI API key into the "OpenAI API Key" field.
4.  **Select Provider Model:** Pick the model you want from the "Model" dropdown.
5.  **(Optional) Base URL:** If you need a proxy or a custom base URL for the OpenAI API, enter it here. Most users won't need to change the default.

### Tips and Notes

-   **Pricing:** Review the [OpenAI Pricing page](https://openai.com/pricing) for detailed cost information across models.
-   **Azure OpenAI Service:** For Azure OpenAI, configure it as an OpenAI-compatible endpoint with the appropriate base URL.

## OpenAI Codex (OAuth)

1. Open Trembo settings.
2. Select **OpenAI Codex** from **API Provider**.
3. Click **Sign in with OpenAI** and complete the browser OAuth flow.
4. Return to Trembo and choose a model.

Notes:
- No API key entry required.
- Available models depend on your OpenAI plan.
