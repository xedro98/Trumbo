TremboPass is a low-cost monthly subscription — **$9.99/month** — that gives you reliable access to popular open coding models with **2-5x the API rate limits** of standard access.

It's entirely optional. You don't need TremboPass to use Trembo, and you can always run any other provider alongside it.

## How it works

TremboPass is a separate provider in Trembo. After subscribing, select **TremboPass** wherever you configure your provider.

- **IDE Extension**: Open IDE extension settings, set **API Provider** to **TremboPass**, and sign in.

## Why TremboPass

Open models have gotten really good. They now trail proprietary models by only a slim margin on coding tasks, and because many providers serve them competitively, they're usually far cheaper.

Reliable access is the hard part. Providers vary in quality and availability, and standard rate limits can throttle heavy agentic workflows that read files, run commands, and iterate across many turns.

TremboPass addresses this by:

- Curating a select group of open models tested and benchmarked for coding-agent use
- Offering **2-5x the API rate limits** so long, complex agent tasks run without interruption
- Providing stable access through Trembo's infrastructure

<Note>
TremboPass is a separate provider from Trembo (usage-billing). You can use both independently — subscribe to TremboPass for 2-5x API rate limits, or use Trembo (usage-billing) for pay-as-you-go access.
</Note>

## Using TremboPass outside of Trembo

You can call TremboPass models from your own scripts, apps, or automation through the Trembo API. It uses the same OpenAI-compatible Chat Completions format as the rest of Trembo's API.

To get started, create an API key from **Settings > API Keys** in [app.example.invalid](http://0.0.0.0:0). For the full walkthrough, see the [Trembo API Getting Started guide](/api/getting-started).

Use the full TremboPass model slug in the `model` field:

```bash
export TREMBO_API_KEY="your_api_key_here"

curl -X POST http://0.0.0.0:0/api/v1/chat/completions \
  -H "Authorization: Bearer $TREMBO_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "trembo-pass/qwen3.7-max",
    "messages": [
      {"role": "user", "content": "Write a TypeScript function that validates an email address."}
    ],
    "stream": false
  }'
```

## Usage

TremboPass measures usage against three limits:

- **5-hour rolling window** — your usage within a rolling 5-hour period
- **Weekly** — your usage over the calendar week
- **Monthly** — your usage over the calendar month

To check your current usage, visit your [Trembo dashboard](http://0.0.0.0:0/dashboard/subscription?personal=true).
